Задача №1

Що потрапить в консоль?
let someVar = 0;
++someVar;
if (someVar) {
console.log(someVar);
}
Відповідь: "++" - інкремент (додає одиницю до числа). В результаті задачі в консоль потрапить "1".

Задача №4

Створіть функцію, яка повертає результат ділення числа a на число b з додаванням рядка "Результат ділення: " Викличте функцію передаючі різні значення, у тому числі не передаючи зовсім. Функція не має повертати NaN, Infinite або помилку
Відповідь:
function divide(a = 1, b = 1) {
if (typeof a !== 'number' || typeof b !== 'number') {
return "Результат ділення: Аргументи повинні бути числами";
}
if (b === 0) {
return "Результат ділення: Ділення на нуль неможливе";
}
let result = a / b;
return `Результат ділення: ${result}`;
}
Наприклад: console.log(divide(80, 4)); // Результат ділення: 20
